2 Way Partial Dependency Plot

Use this application to generate 2 way partial dependency plots for any model produced by a DataRobot project.

Once generated, the plots will be saved in the folder /static and reused if you request them again. The file name will be as follows: 

   PRJECT_ID-MODEL_ID-COLONE_NAME-COLTWO_NAME.png

If you want to regenerate a plot using an alternative dataset you will need to delete the old version.
